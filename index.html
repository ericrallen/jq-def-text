Why use jq-def-text?

jq-def-text aims to provide a simple solution to pre-filling form fields with default values that will take browsers capable of placeholder text and those that aren't into account.

It has a small footprint with a decent amount of functionality so you can configure it to work in whatever way you prefer.

Example Global Option Configurations

If you want to change some options once and have them carry over to all of your jq-def-text instances, you can configure the global default options and not worry about them again.

  • Change Default Class

    //by default jq-def-text applies the class 'iadeftext'
//you could disable the default class by setting the value to '' if you prefer
$.fn.iaDefaultText.default_settings.def_class = 'default';

  • Toggle HTML5 Placeholder Text

    //by default the html5 placeholder attribute leaves the text in the box until the user types something
//you can tell it to remove the placeholder on focus and replace it on blur, if you prefer
$.fn.iaDefaultText.default_settings.toggle_placeholder = true;

  • Turn On Modernizr HTML5 Placeholder Check

    //by default jq-def-text uses Modernizr to check for the placeholder capability
//if you want to use Modernizr to check for browser placeholder functionality, you can tell jq-def-text to check for the capability on it's own. This requires that the build of Modernizr you are using is configured to check input capabilities.
$.fn.iaDefaultText.default_settings.use_modernizr = true;

  • Turn Off HTML5 Placeholder Text

    //by default jq-def-text uses the html5 placeholder attribute where it can
//you can tell it not to if you don't like how unruly styling placeholder text can be
$.fn.iaDefaultText.default_settings.use_placeholder = false;

  • Set a Default Callback Function

    //you can tell jq-def-text to run a callback function when the field is blurred
//you can set a default callback for every field and still override it
//with a field-specific callback when you initialize a jq-def-text field
$.fn.iaDefaultText.default_settings.callback = function() {
    //your callback function here
};

Example Configurations on a Per-Field Basis

Keep in mind you can use any combination of these configurations to tailor jq-def-text to your liking. For our examples, we'll be using an example id selector '#example' but you can use pretty much any selector to call jq-def-text.

  • Default Everything

    $('#example').iaDefaultText({
    def_text : 'Your Name'
});

  • Custom Default Class

    $('#example').iaDefaultText({
    def_text : 'Your Name',
    def_class : 'default'
});

  • Toggle HTML5 Placeholder Text Display on Focus/Blur

    $('#example').iaDefaultText({
    def_text : 'Your Name',
    toggle_placeholder : true
});

  • Don't Use Modernizr to Check HTML5 Placeholder Text Functionality

    $('#example').iaDefaultText({
    def_text : 'Your Name',
    use_modernizr : false
});

  • Don't Use HTML5 Placeholder Text

    $('#example').iaDefaultText({
    def_text : 'Your Name',
    use_placeholder : false
});

  • Trigger Callback Function on Blur

    $('#example').iaDefaultText({
    def_text : 'Your Name',
    callback : function() {
            //your function here
    }
});
